# Project Archive

This directory contains historical specifications, implementation plans,
validation records, and evidence that are no longer authoritative project
direction.

Current direction lives in:

- `docs/design-direction-governance-sandbox.md`
- `docs/plans/2026-07-20-testing-stance-correction.md`
- `docs/architecture.md`

## `market-trading-prototype/`

This archive preserves the authored 20-system market-trading prototype and
its implementation history. It includes the initial specification, completed
and abandoned plans, market/NPC-fleet validation contracts, long-run
metastability evidence, and terminal UI captures.

The archive remains useful for code archaeology and for extracting isolated
mechanisms such as checked Energy accounting. It must not be treated as a
current roadmap, acceptance contract, or compatibility requirement. In
particular, references inside archived documents to “current,” “required,”
“ready,” or “authoritative” describe their historical context only.
