# Lore

Lore pages contain canonical setting context and may retain explicitly
unresolved fiction. Lore does not create a mechanical requirement. A ruin,
faction, resource, route, or historical process exists in gameplay only when a
[current design contract](../current/README.md) defines it.

- [Precursor Aftermath](precursor-aftermath.md) — post-collapse setting, ruined-grid framing, and the origin community.

Direction may use lore to explain an intended experience, but mechanics remain
owned by current design or explicitly committed direction.
