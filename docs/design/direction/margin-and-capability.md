# Margin and Capability

## Margin allocation

The core loop is generating and allocating physical margin around life support and infrastructure pressure. Its durable uses are:

- **bank** — reserve against shocks;
- **develop** — improve present capability; and
- **expand** — commit resources outward.

These verbs need not all exist at tick zero. Current mechanics define the implemented bank, development, and founding contracts; this direction does not turn runway into an unstated solvency oracle or prescribe a particular UI.

## Capability over wealth

A run should accumulate vulnerable capability rather than abstract wealth. Production chains, infrastructure, knowledge, people, and logistics matter because they let the community survive and act. Capability can be lost through the same physical rules that create it.

Specialists, training, tertiary goods, multiple production chains, and exact runway presentation are possible mechanisms, not committed direction. They are kept in [Production and Capability Ideas](../ideas/production-and-capability.md).

The approved current physical-resource substrate is indexed in [Prototype Migration](prototype-migration.md#retained-economic-substrate).
